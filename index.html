HTML Head and Body element

HTML Head element

The HTML head element is a container for the following elements:

  • Title
  • Style
  • Meta
  • Link
  • Script

The head element is a container for metadata and placed between the html and body element.

Title

This element defines the title of the element and should be text only. This title is shown in the browser's title bar.
It is a required element of a HTML document for Search Engine Optimization (SEO). The title content is very important for SEO and try to make it more meaningful.

Style

The style ekement is used to define style information for a single HTML document. It is also used for internal CSS.

Meta

The element is typically used to specify the character set, page description, keywords, author of the document, and viewport settings.

Link

The link element is used to define a relationship between current document and external resources. It is mostly used to link external stylesheet.

Script

The script element is used to define client side JavaScript. Functions between these script element will be executed in the document.
Script element is also used to link between JavaScript file and HTML document.

Any element within the Head element won't be displayed on the page but will be used by the browser and by search engines.
